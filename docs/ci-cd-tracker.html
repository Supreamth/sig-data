How to continue after changing model

Use this if the model changes or you start a new terminal. Hermes keeps sessions in its session database, so you can resume by session ID or continue the most recent session.

Continue most recent sessionhermes --continue

List sessions if unsurehermes sessions list
Change model inside a running chatUse slash command /model, pick the model, then continue in the same session.
